# ErefAI Enhancement Frontend

Frontend aplikacija za upravljanje korisnicima, profesorima, studentima i predmetima.

## 🚀 Tehnologije
- React (Vite)
- Axios
- React Router

## 📌 Funkcionalnosti
- Login (JWT autentifikacija)
- Upravljanje korisnicima (create, delete)
- Upravljanje rolama
- Upravljanje predmetima
- Kreiranje studenata i profesora
- Dodela predmeta profesorima

## 🔐 Autentifikacija
Aplikacija koristi JWT token koji se dobija nakon login-a i čuva u localStorage.

## ▶️ Pokretanje projekta

```bash
npm install
npm run dev
